# ProgressTracker

A progress tracker component is a visual navigation element typically used to display progress or guide user through a multi-step process.

Step indicator content defaults to step index, or checkmark icon when completed. Use `StepIndicator` to customize this content, through `complete` and `incomplete` props.

<Canvas of={stories.Default} />

## Install

```sh
npm install @spark-ui/components
```

## Anatomy

```tsx
import { ProgressTracker } from '@spark-ui/components/progress-tracker'

export default () => (
  <ProgressTracker>
    <ProgressTracker.Step>
      <ProgressTracker.StepIndicator />
      <ProgressTracker.StepLabel />
    </ProgressTracker.Step>
  </ProgressTracker.Step>
);
```

## API Reference

### ProgressTracker

Contains all the progress tracker component parts.

<ArgTypes of={ProgressTracker} />



### _.Step

Contains the button associated to a step.

<ArgTypes of={ProgressTracker.Step} />

### _.StepLabel

The label describing the step.

<ArgTypes of={ProgressTracker.StepLabel} />

### _.StepIndicator

Contains the custom content of a step, for complete and incomplete states.

## Accessibility

<A11yReport of="progress-tracker" />

To enforce your implementation's accessibility we recommend using a specific `aria-label` on `ProgressTracker` root component.

### Keyboard Interactions

- <Kbd>Tab</Kbd>: Moves focus to the next step depending on `orientation`.
- <Kbd>Space</Kbd>: Activates the step.
- <Kbd>Enter</Kbd>: Activates the step.
