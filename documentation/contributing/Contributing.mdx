# Contributing to Spark UI

**Spark UI** is Adevinta's React Design System focused on modularity and accessibility. We welcome contributions from the community to help improve and expand our component library.

## Getting Started

To become a contributor to Spark UI, please contact [an admin](https://github.com/Powerplex) to be added to the contributor group. Once you have access, you can start contributing to the project following our guidelines and best practices.

<div>
  <Callout kind="info" title="Note to contributors from GHE">
    <p>
      Spark is an open-source project and we welcome contributions from the community. 
      However, you must use your personal GitHub account to contribute to the project.

      Read [this helpful guide](/docs/contributing-multiple-github-accounts--docs) to to manage multiple GitHub accounts on your machine.
    </p>
  </Callout>
</div>
