# Third-party libraries

Most Spark components are fully custom, but some of them are using third-party libraries.

This section lists the components that are using third-party libraries, and the libraries they are using.

The mid-term goal is to migrate all component to `Base UI` if possible, as its core set of components is a good base for Spark components, more exhaustive than Radix while keeping a similar approach.
