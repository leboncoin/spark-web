# Third-party libraries

Most Spark components are fully custom, but some of them are using third-party libraries.

This section lists the components that are using third-party libraries, and the libraries they are using.

The mid-term goal is to migrate all component to `Base UI` if possible, as its core set of components is a good base for Spark components, more exhaustive than Radix while keeping a similar approach.

## Stable

| Section                 | Components                                                                                                                                                                                                                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Homemade components** | `Badge`, `Breadcrumb`, `Button`, `Carousel`, `Chip`, `FormField`, `Icon`, `IconButton`, `Input`, `Kbd`, `LinkBox`, `ProgressTracker`, `Rating`, `RatingDisplay`, `ScrollingList`, `Select`, `Skeleton`, `Spinner`, `Tag`, `TextLink`, `Textarea`, `VisuallyHidden` |
| **Using Base UI**       | `Accordion`, `AlertDialog`, `Dialog`, `Drawer` (indirectly using Dialog), `CircularMeter`, `Meter`, `Collapsible`, `Progress`, `Slider`, `Toast`                                                                                                                   |
