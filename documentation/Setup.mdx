# Setup

## Prerequisites

**Spark UI** is built on **Tailwind CSS v4**. [Install Tailwind CSS](https://tailwindcss.com/docs/installation) in your project first.

For a better developer experience (autocomplete, syntax highlighting, linting), use the [**Tailwind CSS IntelliSense**](https://tailwindcss.com/docs/editor-setup#intelli-sense-for-vs-code) extension for VS Code, or similar tools for [JetBrains IDEs](https://www.jetbrains.com/help/webstorm/tailwind-css.html).

Finally, make sure your project is using **React 19** or higher.

## Spark setup

After completing the Tailwind CSS setup, you'll simply have to install the following packages via your command line:

```bash
npm i @spark-ui/theme-utils @spark-ui/components
```

Navigate to your **Tailwind CSS configuration file** and refer to the example below for applying our Spark theme:

```css
@import 'tailwindcss';

@import './node_modules/@spark-ui/theme-utils/dist/style.css'; /* spark themes */

@source './node_modules/@spark-ui'; /* ensure that the Spark components are scanned */
```
