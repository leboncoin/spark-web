# Spark Tokens

## Colors

### Color families

Spark opts out of the default TailwindCSS color palette and offers 9 color families, each with multiple color tokens with their own semantic usage.

- **Main**: Use Main color to emphasize the most relevant components and essential elements on the UI, like primary actions or brand panels. These colors play a crucial role in helping to strengthen the brand identity. By using the Main colors strategically, users can easily recognize and connect with the brand, which leads to building trust and familiarity with an interface that is cohesive and unique.

- **Support**: Use Support colors to complement the main actions or primary elements to provide a sense of hierarchy and visual consistency in the UI. They are typically darker tones that complement the primary color scheme without overpowering it. They help distinguish secondary elements from primary ones, making it easy for users to navigate the interface.

- **Accent**: Use accent color to highlight important elements in the interface. Draw attention to specific elements and create visual interest while guiding users' focus.

- **Success**: Convey to the user a successful action, like after a form has been successfully submitted.

- **Alert**: Convey to the user a warning, for example over a sensitive action.

- **Error**: Convey to the user an error, for example when a form field is invalid.

- **Info**: Convey to the user additional information, for example in a tooltip.

- **Neutral**: Neutral family should be using grey colors and is used for neutral elements, such as: border, dividers, a banner with no specific intent, etc.

### Colors syntax

Each color family offers various tokens with their own semantic usage.

#### Base token

Main color tokens of each family, they represent the visual identity of your theme and should be vivid/strong. They are used for important elements of your interfaces, such as the main call to actions and elements that makes your website identity (logo, illustrations, etc)

```jsx
<div className="bg-main" />
```

#### Container token

Toned down versions of the main color tokens. They must re-use the same color hue but be less saturated or with softer lightness. They are used for background elements of your interfaces.

```jsx
<div className="bg-main-container" />
```

#### Contrasting color token

Colors to use on top of a `*` or `*-container` background. Usually for text or icons. Their purpose is to contrast with the background color.

- Never use as a background.
- Never use on a background of a different color family.
- They don't exist in an `*-hovered` state, as Spark chose to animate the background color instead.

```jsx
<div className="bg-main text-on-main" />
<div className="bg-main-container text-on-main-container" />
```

#### Hovered token

Hovered state of a `*` or `*-container` element. We advise to use the same color hue and saturation as the base colorand simply apply a delta on the lightness.

```jsx
<button type="button" className="bg-main hover:bg-main-hovered">
  Click me
</button>
```

#### Outline token

Colors to use for generic outlines/dividers. **Never use on text or background**.

```jsx
<hr className="border-sm border-outline" />
```

#### Outline high token

Colors to use for high contrast outlines/dividers. **Never use on text or background**.

```jsx
<hr className="border-sm border-outline-high" />
```

#### Overlay token

Colors to use for overlays (ex: modals, tooltips, dropdowns). Combine it with `dim-3` opacity for consistancy.

```jsx
<div className="bg-overlay/dim-3" />
```

## Shadows

### Regular shadows

Use utilities like `shadow-xs` and `shadow-sm` to apply a box shadow to an element. You can prefix it with `inset-` to apply it to the inside of the element.

You can also have both an inset and a regular shadow at once.

### Ring shadows

Use utilities like `ring-2` and `ring-4` to apply a solid box-shadow to an element. You can prefix it with `inset-` to apply it to the inside of the element.

You can combine a ring with a shadow to create a more complex effect. You can also have both an inset and a regular ring at once.

### Colored shadows

Use utilities like `shadow-main` (or any other Spark color) to change the color of a box shadow.

Use utilities like `ring-main` (or any other Spark color) to change the color of a ring.
