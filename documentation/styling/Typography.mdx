# Typography

## Denim INK Font

Spark UI includes the **Denim INK** font family, a custom typeface designed specifically for the design system. It comes with four weights to support various typographic hierarchies:

- **Regular (400)** - For body text and standard content
- **Medium (500)** - For subtle emphasis
- **Semi-Bold (600)** - For subheadings and moderate emphasis
- **Bold (700)** - For headings and strong emphasis

## Setting up the font in your project

<Callout variant="warning">
  **Important**: Spark UI does not bundle font files in the npm package. You must host and load the Denim INK font yourself.
</Callout>

The Denim INK font files are available in the Spark repository under [`assets/fonts/lbc-denim-ink/`](https://github.com/leboncoin/spark-web/tree/main/assets/fonts/lbc-denim-ink).

### Option 1: Host on your CDN (Recommended)

1. **Upload font files** to your CDN
2. **Add `@font-face` declarations** to your CSS:

```css
/* Add before importing Spark theme */

@font-face {
  font-family: 'LBC Denim INK';
  src: url('https://your-cdn.com/fonts/LBCDenimINK-Regular.woff2') format('woff2'),
       url('https://your-cdn.com/fonts/LBCDenimINK-Regular.ttf') format('truetype');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'LBC Denim INK';
  src: url('https://your-cdn.com/fonts/LBCDenimINK-Medium.woff2') format('woff2'),
       url('https://your-cdn.com/fonts/LBCDenimINK-Medium.ttf') format('truetype');
  font-weight: 500;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'LBC Denim INK';
  src: url('https://your-cdn.com/fonts/LBCDenimINK-SemiBold.woff2') format('woff2'),
       url('https://your-cdn.com/fonts/LBCDenimINK-SemiBold.ttf') format('truetype');
  font-weight: 600;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'LBC Denim INK';
  src: url('https://your-cdn.com/fonts/LBCDenimINK-Bold.woff2') format('woff2'),
       url('https://your-cdn.com/fonts/LBCDenimINK-Bold.ttf') format('truetype');
  font-weight: 700;
  font-style: normal;
  font-display: swap;
}

/* Then import Spark theme */
@import 'tailwindcss';
@import '@spark-ui/theme-utils/base';
```

### Option 2: Self-host in your project

1. **Copy font files** from `assets/fonts/lbc-denim-ink/` to your `public/fonts/` directory
2. **Add `@font-face` declarations** with local paths:

```css
@font-face {
  font-family: 'LBC Denim INK';
  src: url('/fonts/lbc-denim-ink/LBCDenimINK-Regular.woff2') format('woff2'),
       url('/fonts/lbc-denim-ink/LBCDenimINK-Regular.ttf') format('truetype');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

/* Repeat for weights 500, 600, 700 */
```

3. **Import Spark theme**:

```css
@import 'tailwindcss';
@import '@spark-ui/theme-utils/base';
```

## Font loading characteristics

- **Format**: WOFF2 (primary) and TTF (fallback) for optimal browser support
- **Loading strategy**: `font-display: swap` to prevent invisible text during loading
- **File size**: Optimized web fonts (~80KB for WOFF2, ~215KB for TTF per weight)

## Font stack

The complete font stack includes system font fallbacks to ensure text remains readable even if the custom font fails to load:

```css
font-family: 'LBC Denim INK', ui-sans-serif, system-ui, -apple-system, 
  BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 
  'Noto Sans', sans-serif;
```

## Typography utilities

Spark provides pre-defined typography utilities that use the Denim INK font with appropriate weights and sizes. Use these classes for consistent typography:

```jsx
<h1 className="text-display-1">Display heading</h1>
<h2 className="text-headline-1">Section heading</h2>
<p className="text-body-1">Body text</p>
<small className="text-caption">Caption text</small>
```

See the [Design Tokens](?path=/docs/styling-design-tokens--docs) page for the complete list of typography utilities.

## Font weights

Access different font weights using Tailwind's font-weight utilities:

```jsx
<p className="font-regular">Regular text (400)</p>
<p className="font-medium">Medium text (500)</p>
<p className="font-semibold">Semi-bold text (600)</p>
<p className="font-bold">Bold text (700)</p>
```

## Using a custom font

If you need to replace Denim INK with your own custom font:

1. **Add your font files** to your project (e.g., in a `public/fonts/` directory)

2. **Define `@font-face` declarations** in your CSS:

```css
@font-face {
  font-family: 'Your Custom Font';
  src: url('/fonts/your-font-regular.woff2') format('woff2'),
       url('/fonts/your-font-regular.woff') format('woff');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

/* Repeat for other weights... */
```

3. **Override the `--font-sans` variable** in your Tailwind config:

```css
@import 'tailwindcss';
@import '@spark-ui/theme-utils/base';

@theme {
  --font-sans: 'Your Custom Font', ui-sans-serif, system-ui, sans-serif;
}
```

<Callout variant="warning">
  When using a custom font, ensure you include all necessary font weights (400, 500, 600, 700) to maintain the design system's typographic hierarchy.
</Callout>

## Performance considerations

To optimize font loading performance:

- **Preload critical fonts** in your HTML `<head>`:

```html
<link rel="preload" href="/path/to/font.woff2" as="font" type="font/woff2" crossorigin>
```

- **Use `font-display: swap`** to show fallback fonts while custom fonts load
- **Subset fonts** if you only need specific character sets
- **Serve fonts from a CDN** for better caching and geographic distribution

## Accessibility

The Denim INK font has been designed with readability in mind:

- Clear letterforms with good x-height
- Adequate spacing between characters
- Multiple weights for proper hierarchy
- Works well at various sizes from small (caption) to large (display)

Always ensure sufficient color contrast ratios when using different font weights. Refer to the [Themes](?path=/docs/styling-themes--docs) documentation for color accessibility guidelines.
